toggl.vim
=========

Toggl client for vimscript and vim.

Usage
-----
Start task

```vim
:TogglStart task name
```

Stop current task

```vim
:TogglStop
```

unite.vim interface
--------------------

There are two sources:

```vim
:Unite toggl/task
```

which helps you to restart past time entries.

```vim
:Unite toggl/project
```

which helps you to change the project of current task.
